# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from raw data processing to business insights visualization. The project involves data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis using MySQL, interactive dashboard creation in Power BI, and presentation of findings through a report and presentation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

* Dataset: Customer Behavior Dataset
* Format: CSV
* Records: Customer transaction and behavioral data
* Features include:

  * Customer ID
  * Age
  * Gender
  * Purchase Information
  * Product Category
  * Payment Method
  * Review Ratings
  * Frequency of Purchases
  * Other customer-related attributes

## Project Workflow

### 1. Data Loading

* Imported the dataset using Python.
* Examined dataset structure, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends.
* Identified patterns and relationships.
* Generated summary statistics and visualizations.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected data inconsistencies.
* Prepared data for analysis and reporting.

### 4. SQL Analysis

* Imported cleaned data into MySQL.
* Executed SQL queries to answer business questions.
* Performed aggregations, filtering, ranking, and trend analysis.

### 5. Dashboard Development

* Connected data to Power BI.
* Created interactive dashboards and visual reports.
* Designed KPI cards, charts, filters, and slicers.

### 6. Reporting & Presentation

* Prepared a detailed analytical report.
* Created a professional presentation using Gamma AI.
* Summarized key findings and recommendations.

## How to Run

### Python Analysis

1. Clone the repository.
2. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook or Python files.
4. Run the notebooks in sequence:

   * Data Loading
   * Data Cleaning
   * EDA
   * Analysis

### MySQL Analysis

1. Create a database in MySQL.
2. Import the cleaned dataset.
3. Execute the SQL queries available in the `SQL` folder.

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the dataset connection.
3. Explore dashboard visuals and insights.
